## Download Terraform
https://www.terraform.io/downloads
</br>

## Building and Running
1. Install the dependencies and initialize the project
`terraform init`

2. Code and configure the resources for the plan 
`terraform plan`

3. Verify plan and run
`terraform apply`
</br>
